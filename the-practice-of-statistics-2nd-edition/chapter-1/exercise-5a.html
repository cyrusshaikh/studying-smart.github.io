Exercise 5a

Problem:

The table below shows the percentage of females earning doctorates in different fields of study. Use this data to create a bar graph.

Field of Study Percentage of Females Earning Doctorates
Computer Science15.4%
Education60.8%
Engineering11.1%
Life Sciences40.7%
Physical Sciences21.7%
Psychology62.2%

Solution:

We can represent this data using a bar graph. In the bar graph, each field of study will be displayed along the horizontal axis. The height of each bar corresponds to the percentage of females earning doctorates in that field. This visual representation allows for an easy comparison between fields.
